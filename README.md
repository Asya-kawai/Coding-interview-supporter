# Coding-interview-supporter
A service that supports coding interviews.

## 本サービスが目指すところ

* 考え方の可視化

## 既存サービスの課題

既存サービスが提供する機能は大まかに以下の2点。

* 基本的知識のチェック
* 問題に対して答えを算出するコードとそのチェック

コーディング面接は、以下の点をチェックするべき。

* 考え方・・・
* 経験・・・
* 組織文化との相性・・・
* コーディングスキル
* 基本知識

なお、コーディングスキルと基本知識については既存サービスとの同質化すべき点であることに留意。
